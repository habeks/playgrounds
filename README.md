# AR + Map MVP

Маленькое веб-приложение с:
- Камерой (AR фон)
- 3D кубом на Three.js
- Leaflet-картой с геопозицией
- Кнопкой «Сканировать»
- Инвентарём

## Как запускать

1. Зайди в репозиторий
2. Открой Codespace
3. Открой `index.html` через Live Preview или GitHub Pages
